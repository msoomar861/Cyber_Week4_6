# Cyber Security Internship Project

This project demonstrates secure API development and cybersecurity best practices.

## Features
- Rate Limiting
- API Key Authentication
- CSRF Protection
- SQL Injection Prevention
- Security Headers (CSP & HSTS)

## Tools Used
- Node.js
- Express
- OWASP ZAP
- Burp Suite
- SQLMap

## How to Run
1. Install dependencies
2. Add API key in .env
3. Run server using node app.js
